■将棋ウォーズ棋譜ダウンローダ


将棋ウォーズの棋譜を一括ダウンロードするWindows用のソフトです。

-----------------------------------------------------------
■制限事項


・Windows専用です
・自分の棋譜のみダウンロードできます
・直近30日の棋譜のみダウンロードできます
・指導対局には対応していません

-----------------------------------------------------------
■使い方


1. 「id.txt」に自分の将棋ウォーズIDを記述してください

2. IEで将棋ウォーズにログインしてください。https://shogiwars.heroz.jp/web_app/standard/
   ログイン済みならこの作業は不要です

3. 「将棋ウォーズ.js」を実行してください


棋譜はKIF形式で保存されます。
将棋GUIなどの将棋ソフトで開けます。http://shogigui.siganus.com/
